Website Analisis Tren Harga Saham

Website tren harga saham menggunakan Streamlit untuk analisis tren harga saham dengan regresi linier. Data diinput, diproses dengan Pandas dan NumPy, lalu dianalisis menggunakan model Scikit-learn. Hasil prediksi divisualisasikan dengan Matplotlib atau Plotly, memungkinkan pengguna berinteraksi secara real-time. Backend dan frontend dikelola dalam app.py, sementara hosting dilakukan melalui Streamlit Cloud untuk akses online tanpa konfigurasi server manual.
